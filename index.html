🎵 ESP32 Music System

Benjamin & Kfir

This project explores how microcontrollers can recreate real music using buzzers.

The Idea

We wanted to go beyond simple buzzers. Instead of just making noise, we asked:

Can an ESP32 recreate real songs?

Hardware Upgrade

We switched from Arduino to ESP32 for:

• Faster processing • Better timing precision • More flexibility for future features

How It Works

Each musical note is converted into a frequency. The ESP32 sends signals to buzzers to reproduce those frequencies.

Code Complexity

Instead of simple tones, we created a structured system:

• Notes mapped to frequencies • Timing controlled precisely • Multiple buzzers synchronized
